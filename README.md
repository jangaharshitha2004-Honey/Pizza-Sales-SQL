📌 Project Synopsis

The goal of this project is to utilize SQL to analyze pizza sales data in order to derive valuable insights that can enhance business choices. In order to comprehend consumer ordering trends, revenue creation, and product performance, the analysis entails querying a pizza sales database.

This project's primary goal is to use SQL queries to turn unprocessed sales data into useful insights. This study handled a number of analytical problems, including figuring out the most popular pizza varieties, calculating total income, examining order trends over time, and comprehending the distribution of sales by category.

This project uses SQL to demonstrate actual data analytics skills, such as data querying, data aggregation, data combining, and business insight production.

📂 Dataset Description

The dataset contains information about pizza sales including:
	•	Orders – Contains order ID and order date/time
	•	Order Details – Includes pizza quantities ordered per order
	•	Pizzas – Contains pizza size and price information
	•	Pizza Types – Includes pizza name and category

These tables are connected using primary keys and foreign keys, allowing complex queries using JOIN operations.

📊 Business Questions Solved

This project answers several important business questions related to pizza sales.

1️⃣ Basic Analysis
	•	Retrieve the total number of orders placed
	•	Calculate the total revenue generated from pizza sales
	•	Identify the highest-priced pizza
	•	Identify the most commonly ordered pizza size
	•	List the top 10 most ordered pizza types


2️⃣ Intermediate Analysis
	•	Join necessary tables to calculate the total quantity of each pizza category ordered
	•	Determine the distribution of orders by hour of the day
	•	Find the category-wise distribution of pizzas
	•	Calculate the average number of pizzas ordered per day


3️⃣ Advanced Analysis
	•	Determine the top 5 most ordered pizza types based on revenue
	•	Calculate the percentage contribution of each pizza type to total revenue
	•	Analyze the cumulative revenue generated over time
	•	Determine the top 10 most ordered pizzas based on revenue for each category


📈 Key Insights

Using SQL queries, this analysis helps identify:
	•	Customer ordering patterns
	•	Peak ordering hours
	•	Best-selling pizza categories
	•	High revenue generating pizza types
	•	Sales distribution across time

These insights can help businesses improve menu strategy, pricing, and inventory management.
